# Prescription Tracker

A open-source web application to help you keep track of your prescriptions and never miss a re-order date. Built with React, TypeScript, and Firebase.

## **NOTE**
This tool is in active development. **Do not** rely on it for medical needs as it may contain bugs or errors. Data accuracy and permanence cannot yet be guaranteed.
